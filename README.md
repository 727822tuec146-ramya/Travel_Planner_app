# Travel Planner Application

This is a web application built using Node.js and JavaScript. The project is designed to manage and organize travel-related data with a structured backend.

## Features
- Organized backend structure using MVC pattern
- API handling using routes and controllers
- Data management using models
- Clean and modular code structure

## Technologies Used
- Node.js
- JavaScript
- Express.js
- MySQL (if used)

## Project Structure
- controllers/
- models/
- routes/
- middlewares/
- utilityMethods/

## How to Run
1. Install dependencies:
   npm install

2. Start the server:
   npm start
